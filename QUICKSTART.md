# Quick Start Guide

## 1. Install Dependencies

```bash
npm install
```

## 2. Start the Server

```bash
npm start
```

## 3. Open in Browser

Visit `http://localhost:5000`, or use the URL printed in the terminal.

## 4. Start Using

- Click **+ New** to create a saved project.
- Enter the jewelry pricing details.
- Click **Save Project** to save the project in this browser.

Projects are saved locally in the browser. No account or login is required.
