# Jewelry Pricing Calculator

A public web-based jewelry pricing calculator for estimating jewelry production cost, wholesale price, retail price, and profit.

## Quick Start

```bash
npm install
npm start
```

Then open `http://localhost:5000` in your browser, or use the URL printed in the terminal.

## Features

- No login required
- Save projects in the current browser
- Live metal spot pricing for gold, silver, and platinum
- STL and mesh-based 3DM volume scanning
- Production, wholesale, retail, tax, and profit calculations
- Responsive desktop and mobile layout

## Storage

Projects are saved in browser `localStorage`. They stay on the same browser/device unless browser data is cleared.
